# Hero illustration

`shipday-workspace.png` here is a **placeholder**, not the final art. It is a
1600 x 900 (16:9) PNG generated by `scripts/gen-hero-placeholder.mjs`, drawn as
a dark operations-room backdrop with the standard placeholder frame and cross
so it is obviously not finished art.

## Swapping in the final illustration

Replace this file with the final illustration at the same path and the same
16:9 aspect ratio. No code change is needed: the hero references
`/hero/shipday-workspace.png` as a static poster, and the WebGL scene composites
over it. Keep the file reasonably optimized (a few hundred KB or less) so it
stays the landing's friendly Largest Contentful Paint image.

Recommended export: 1600 x 900 or 2400 x 1350, PNG or WebP renamed to the same
`.png` path is fine since the markup sets the type by extension. Update the
`alt` text in `components/hero/Hero.tsx` if the final scene differs from the
described one.
